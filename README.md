# Refuge Documentation

http://docs.refuge.io


## Requirements

- Python 2.x or sup
- [Pip](https://pypi.python.org/pypi/pip) 
- [SPHINX](http://sphinx-doc.org/)

How to contributre?
-------------------

- Install sphinx:

    $ pip install sphinx

- Compile the documentation:

    $ make htmk and open the generated documentation in the ` _build`
folder.
